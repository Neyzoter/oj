Question :

Need for Speed
 
You are in an illegal racing tournament with no rules. The first one to reach the final position wins $100K. There are multiple roads passing through the different cities. Some cities may contain nitro boosters, sand grounds, crash sites or cops.

The time to reach from one city to another is 1 hour. Nitro boosters increase your speed by 2 times for 2 cities. Sand grounds reduce your speed to half for 2 cities. Crash sites are those cities where crashes happen very often and hence, it takes 1 hour to safely drive out of a crash site. If you meet a cop car in any city, you are busted and lose automatically.
 
Given number of cities for the tournament and roads between cities, find out the best path that requires the least time. Cities are named from 0 to N-1 where N is the number of cities. City 0 is always the starting city and City N-1 is always the final destination. 
 
Note:The calculations on speed for each city will be done on current speed and not the initial speed.
 
Input Specification:
input1: Number of cities.
input2: Array representing special powers associated with the city ["Nitro", "Sand", "Crash", "Cop", "None"] where the ith element is the power present in the ith city.
input3: Number of roads.
input4: 2D array representing roads of form {c1, c2} where c1 is the starting city and c2 is the ending city.
 
Output Specification:
Your function must return the path in the form {c1, c2, ..., cn} where c1, c2,..., cn are the cities the car will go through in sequence. Return {-1} if no possible path exists.
 
Example 1:
input1: 5
input2: {"None", "Cop", "None", "None", "None"}
input3: 5
input4: { { 0, 1}, {0, 2}, {1, 2}, {2, 3}, {3, 4 } }
 
Output: {0, 2, 3, 4}
 
Explanation:
2 routes-
1. 0->1->2->3->4
This route cannot be taken as city 1 has cops.
2. 0->2->3->4
This route can be used. As the time required to travel from one city to another is 1 hr,
so the total time taken will be (1+1+1) = 3 hrs.
 
Example 2:
input1: 7
input2: {"None", "Cop", "Sand", "None", "Nitro", "None", "None"}
input3: 8
input4: { { 0, 1}, {0, 2}, {1, 2}, {2, 3}, {2, 4}, {3, 6}, {4, 5}, {5, 6 } }
 
Output: {0, 2, 4, 5, 6}
 
Explanation:
3 routes-
1. 0->1->2->3->6
This route cannot be taken as city 1 has cops.
2.0->2->3->6
Since city 2 has sand, speed will be halved and the time required will be doubled for next 2 cities. Hence, the total time taken will be (1+1*2+1*2) = 5 hrs.
3. 0->2->4->5->6
Since city 2 has sand, speed will be halved and the time required will be doubled  for next 2 cities. But, because there is nitro in city 4, so speed will be doubled and the time required will be halved for next 2 cities Hence, total time taken will be
(1+1*2+1*2*0.5+1*0.5) = 4.5hrs.
 Hence, route 3 ( 0->2->4->5->6) is selected.
