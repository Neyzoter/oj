# 回文数问题（PalindromeNumber）
## 1.问题
判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

```
输入: 121
输出: true
```

示例 2:

```
输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
```

示例 3:

```
输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
```

## 2.思路
**数字倒序**

1、排除负数的情况

2、将数字倒过来

从个位开始加，不断乘10,再加高位。

```
while(x_temp!=0) {
	temp = temp * 10 + x_temp%10;
	x_temp = x_temp / 10;
}

```

3、比较前后是否相等
